React-组件传值

发表于 2020-08-08

一、父传子

　　1、在 state 中准备要传递给子组件的数据

　　2、以属性的方法将值传递给子组件使用的标签位置

　　3、在子组件内部使用 this.props 接收父组件传递过来的数据

二、子传父

　　1、在父组件中声明一个回调函数

　　2、将父组件中声明的函数当成 props 属性值传递给子组件

　　3、子组件通过 props 调用回调函数

　　4、将子组件的数据作为参数传递给回调函数

三、兄弟传值

　　兄弟之间的传值就是：首先 子组件1 向 父组件 传值，然后 父组件 传值给 子组件2
